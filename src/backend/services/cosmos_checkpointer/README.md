To Use the Cosmos Checkpointer Service
======================================
1. Create a CosmosDB account
    - Need 2 collections: 
    - - `checkpoint` 
    - - `checkpoint_writer`
2. Set the environment variables
    - `COSMOSDB_ENDPOINT`
    - `COSMOSDB_KEY`
    - `COSMOSDB_DATABASE`
    - `COSMOSDB_CHECKPOINT_COLLECTION`
    - `COSMOSDB_CHECKPOINT_WRITER_COLLECTION`
